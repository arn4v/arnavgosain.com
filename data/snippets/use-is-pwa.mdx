This hook makes use of the [useMediaQuery](/snippets/use-mediaquery) hook.

**Code**:

```ts
import useMediaQuery from "./use-mediaquery";

export default function useIsPwa() {
  const isPwa = useMediaQuery("(display-mode: standalone)");

  return isPwa;
}
```

**Usage**:

```tsx
import useIsPwa from '../hooks/use-ispwa';

const Component = () => {
  const isPwa = useIsPwa()
  return isPwa ? /** PWA-only component */ : /** Non-PWA component */;
}
```
